> 本系统是一个用SpringBoot+JPA打造的基于微信服务号平台的点餐系统的后端，前端是一个使用Vue.js构建的运行于微信公众号的WebApp。前后端完全分离，前后端之间通过RESTful风格的接口相连。

### 🏠 [Homepage](https://github.com/gb145234/SpringBoot-sell)

## Install

```sh
npm install
```

## Usage

```sh
npm run startstart
```

## Run tests

```sh
npm run test
```
